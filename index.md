# Welcome to My GitHub Pages Site

This is a sample template page for GitHub Pages.

## Features

- Built with Jekyll
- Responsive design
- Easy to customize

## Getting Started

1. Clone this repository
2. Run `bundle install` to install dependencies
3. Run `bundle exec jekyll serve` to preview locally
4. Visit `http://localhost:4000` in your browser

## Customization

Edit `_config.yml` to change site settings, and modify this file to update the homepage content.
